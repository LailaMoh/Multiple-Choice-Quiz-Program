 
# Multiple-Choice Quiz Program

This is a simple Java console program that asks the user five multiple-choice questions and calculates the final score as a percentage.

## Features
- Asks 5 Java-related multiple-choice questions  
- Uses if and switch statements  
- Uses Scanner for user input  
- Converts input to uppercase  
- Displays total correct answers and final percentage  

## How to Run
1. Open the file `QuizGame.java` in a Java IDE or terminal.
2. Compile the program:
   javac QuizGame.java
3. Run the program:
   java QuizGame
4. Answer the questions by typing A, B, C, or D.
